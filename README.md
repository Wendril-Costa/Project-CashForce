## Installation

### Front-End

  1. `cd app`
  2. `cd frontend`
  3. `bash`
  4. `nvm use 16`
  5. `zsh`
  6. `npm install`
  7. `npm start`
  
### Back-End

  1. `cd app`
  2. `cd backend`
  3. `bash`
  4. `nvm use 16`
  5. `zsh`
  6. `npm install`
  7. Copiar o SQL do banco de dados na pasta `backend/db.cashforce.sql`
  8. Configurar seu Sequelize e Banco de dados na pasta `backend/src/database/config/config.js`
  9. `npm start`
